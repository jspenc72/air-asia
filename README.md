# air-asia

# Development

Install dependencies

```bash
npm i
```

Start development server

```bash
ng serve
```

# Deployment

```bash
ng build --extractCss=true --optimization=false
```

```bash
docker build -t air-asia .
```

```bash
docker run air-asia -p 4200:4200 
```
